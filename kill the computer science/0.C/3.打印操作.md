这个本质来讲是上个笔记的补充，来练习一下printf指令。
默认我们打印需要这样写
``` c 
#include <stdio.h>

int main(){
  printf("Hello world!\n");
}
```
``` 输出
Hello world!
```
基本语法就是
``` c
printf("内容");
```
教程里面的练习例子是拿字符画一个三角，也就是。
``` c 
#include <stdio.h>

int main(){
  printf("   /|\n");
  printf("  / |\n");
  printf(" /  |\n");
  printf("/___|\n");
}
```
这里就得注意，程序是自上而下运行的，调换位置会产生不同的效果。
``` output
   /|
  / |
 /  |
/___|
```
这个时候就得问了，怎样输出\呢，由于\在C中是转义符的起始，直接这么写编译会出错，所以我们要用两个，也就是\\\\来表示\(我也是醉了，markdown要用四个来表示两个\\\\)
``` c
#include <stdio.h>

int main(){
  printf("   /\\\n");
  printf("  /  \\\n");
  printf(" /    \\\n");
  printf("/______\\\n");
}
```
``` output
   /\
  /  \
 /    \
/______\
```
